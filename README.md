## Setup

### Frontend
install dependencies and start app using yarn
```bash
# install dependencies (doesn't include quarkus atm)
yarn install
# start nuxt frontend
yarn client:startdev
```
for more nuxt stuff refer to [the client readme](./client/README.md)
